# Chetan Personal Site

A static first-pass personal website for `chetangent.com`.

## Run locally

```bash
python3 -m http.server 4174
```

Then open `http://127.0.0.1:4174/` from this folder.

## Current sections

- Hero / intro
- About
- Relevant links
- Future expansion ideas
